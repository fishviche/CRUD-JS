# Express with MongoDB
App with express and MySQL as DB.

## Requirements
- MySQL
- Node
- Docker (If you want)
## To test
Create a .env file with provided variables 
```sh
MYSQL_HOST='xxxxxxxx'
MYSQL_USER='xxxxxx'
MYSQL_PWD='xxxxxxxx'
MYSQL_DB='xxxxx'
MYSQL_PORT='xxxxx'
PORT='xxxx'
```
And run:
```sh
node index.js
```
